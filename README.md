This is a small React app (Vite) version of the Valentine page with animated hearts and a evasive "No" button.

Quick start

```bash
# install deps
npm install

# run dev server
npm run dev

# build for production
npm run build

# preview production build
npm run preview
```

Deployment

- This project is Vercel-friendly. Connect the repository to Vercel and set the build command to `npm run build` and the output directory to `dist` (Vite default).
